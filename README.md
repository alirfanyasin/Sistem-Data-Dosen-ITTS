# Aplikasi Sistem Data Dosen ITTS
Aplikasi sistem data dosen ITTS merupakan aplikasi yang di rancang untuk mengelolah data informasi dari dosen yang ada di ITTS.
## Features
- Read data 
- Create data
- Edit data
- Update data
- Search data
- Sorting data (Ascending & Descending)

## Main Files
```text
your_folder/
├── main.c
└── function.c
```
